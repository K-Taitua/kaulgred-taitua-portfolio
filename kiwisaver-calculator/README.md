# KiwiSaver Lump Sum Investment Calculator

## Overview

The KiwiSaver Investment Calculator is a Python program designed to estimate the growth of a one-off lump-sum contribution over a selected investment period.

The program uses compound interest calculations with annual return rates of 4%, 6%, and 8% to project potential investment growth.

## Features

- Calculates projected KiwiSaver growth
- Compares 4%, 6%, and 8% return rates
- Optional year-by-year breakdown
- Input validation and error handling
- Formatted summary tables

## Technologies Used

- Python
- Visual Studio Code

- ## How to Run

1. Download or clone this repository.
2. Open `kiwisaver_calculator.py` in Visual Studio Code or another Python editor.
3. Run the programme using Python.
4. Follow the on-screen prompts.
